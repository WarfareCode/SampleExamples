HPOP Force Model Plugin Example:


Step by step procedures for building and using the HPOP Force Model Plugin samples are located in the STK install directory under CodeSamples\Extend\HPOP.ForceModels. There are samples provided for scripting as well as C# and C++ inside the WSC, CSharp, and CPP folders respectively. Please view the corresponding readme files located in each of these directories which will walk you through setting up building Engine Model Plugins.

Required STK Products

STK Expert Edition or STK Professional Edition
