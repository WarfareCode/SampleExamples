Astrogator EOM Function Plugin Example:


Step by step procedures for building and using the Equations of Motion (EOM) Function Plugin samples are located in the STK install directory under CodeSamples\Extend\Gator.EOMFunctions. There are samples provided for scripting as well as C#, C++, and VB.NET inside the WSC, CSharp, CPP, and VB.NET folders respectively. Please view the corresponding readme files located in each of these directories which will walk you through setting up building EOM Functions Plugins.

In this example, an EOM Func plugin is used to give the derivative of four user variables: effective impulse, and the x-, y-, and z-components of integrated delta-V. The user variables are defined on the Mission Control Sequence (MCS) Options panel for the example satellites. The maneuver segment in each example MCS uses a custom propagator, each of which includes an EOM Func plugin in its list of propagation functions. The frame in which the delta-V is integrated is registered as a property of the plugin that can be changed on the propagator panel.

The example EOM plugin registers that it takes the thrust vector and the total mass as input, and that it will give the derivatives of the effective impulse and the integrated delta-V components as output. In its Calc method, it computes the derivative of effective impulse as the thrust magnitude, and it computes the derivative of the integrated delta-V as the thrust acceleration rotated to the specified integration frame.

After each MCS has been run, user value calculation objects can be used to report the user variable values. The report style included in this scenario, "user maneuver vals", can be used to report the effective impulse and integrated delta-V over time.
