Name: CustomApp_AWT_STK_Astrogator_HohmannTransferUsingTargeter
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the code needed to create a Hohmann Transfer orbit using the Astrogator's targeting capabilities.
Directions:
  1. Using the mouse, left click on the buttons in the right hand action pane as they become enabled from top to bottom.
  2. Only click on the Close Scenario button if you wish to start over at any point.
  3. When you reach the Display Results notice the results of the Astrogator propagator are displayed in the message viewer panel in the lower left corner of the application.
  4. You can save or clear this message viewer's text if you wish by clicking on the message viewer toolbar at the left of the message viewer.
  5. Refer to the source code of the application and view the SampleCode.java file if you are interested in the code operations executed when each button is clicked.
  6. Finally click the Close Scenario button.
