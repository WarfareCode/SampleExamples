Name: CustomApp_AWT_STK_X_Map_View
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the sample code and a possible user interface needed to change the view of the STKX Map control (i.e. agi.stkx.awt.AgMapCntrlClass).
Directions: 1. Press the Scenario -> New Scenario menu item.
2. Zoom in on the map control by either clicking the Sample -> Map View -> Zoom In menu item or pressing the Map View Tool Bar's Zoom In button, then:
    a. Click the left mouse button down on the map without releasing.
    b. Move the mouse left to right and then top to bottom.
    c. Note the zoom in rectangle box that is drawn as you drag.
    d. Release the left mouse button up.
    e. Note the map is now zoomed into the region you drew with the zoom in rectangle.
3. Zoom out on the map control by either clicking the Sample -> Map View -> Zoom Out menu item or pressing the Map View Tool Bar's Zoom Out button, then:
    a. Note the map zooms out to the last zoomed in rectangle box.
4. Allow Panning of the map control by either checking the Sample -> Map View -> Allow Pan< menu item or pressing the Map View Tool Bar's Allow Pan button, then:
    a. Click the left mouse button down on the map without releasing.
    b. Move the mouse left, right, up or down.
    c. Note the map pans in the direction you drag.
    d. Note if zoomed out completely, only panning to the left or right is possible.
5. Press the Scenario -> Close Scenario menu item.
