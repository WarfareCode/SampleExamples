Name: CustomApp_AWT_STK_X_Map_Events
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the sample code needed to receive Map control events (i.e agi.stkx.awt.AgMapCntrlClass).
Directions: 1. Press the Scenario -> New Scenario menu item.
2. Move the mouse left to right and top to bottom on the Map control.
3. Click, double click with the mouse left, middle, right button on the Map control.
4. Rotate the mouse wheel if your mouse has one on the Map control.
5. Type on the keyboard while the Map control has focus.
6. Note the events in the message viewers.
7. Press the Scenario -> Close Scenario menu item.
