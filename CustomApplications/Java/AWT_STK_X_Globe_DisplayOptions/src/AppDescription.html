Name: CustomApp_AWT_STK_X_Globe_DisplayOptions
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the basic code needed to change the display options in the STKX Globe control (i.e. agi.stkx.awt.AgGlobeCntrlClass).
Directions:
  1. In the right hand panel, locate the Background pane. Left mouse click on the color button and choose your favorite color. Click ok.
  2. Notice that the background color of the Globe control has changed to your selected color.
  3. In the right hand panel, locate the Splash pane. Left mouse click on the User Defined checkbox to enable it.
  4. If you prefer, choose a new Splash file by left mouse clicking on the Browse button and choosing a file.
  5. Left mouse click on the Splash Panel's Update button.
  6. Notice that the splash screen in the center of the Globe control has changed to your choosen image.
  7. In the right hand panel, locate the Progress Image pane. Left mouse click on the Use User Defined radio button to select it.
  8. If you prefer, choose a new Animate Gif progress image file by left mouse clicking on the Browse button and choosing a file.
  9. Left mouse click on the Splash Panel's Update button.
  10. With the mouse, choose the Scenario -> New Scenario menu item.
  11. Notice that the scenario loading progress animated gif in the Globe control is now using the animated gif you choose.
  12. With the mouse, choose the Scenario -> Close Scenario menu item.
