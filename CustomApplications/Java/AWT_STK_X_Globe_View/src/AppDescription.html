Name: CustomApp_AWT_STK_X_Globe_View
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the code needed to change the view of the STKX Globe control.
Directions: 1. Press the Scenario -> New Scenario menu item.
2. Click the Zoom In button with image of a magnifying glass and plus sign.
3. Click the Home button with image of a house.
4. Press the Scenario -> Close Scenario menu item.
