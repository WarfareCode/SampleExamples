Name: CustomApp_AWT_STK_X_Animation
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the code needed to change animation state via the STKX application.
Directions:
  1. Click on the Scenario -> New Scenario menu item.
  2. Notice the initial Time, Epoch, Frame Rate, and State values.
  3. Click the Play Forward button in the animation tool bar in the upper left corner of the application.
  4. Notice the change in the Time, Epoch, Frame Rate and State values.
  5. Click the Pause button in the animation tool bar in the upper left corner of the application.
  6. Notice the Time, Epoch, Frame Rate and State values have stopped changing.
  7. Click on the Scenario -> Close Scenario menu item.
