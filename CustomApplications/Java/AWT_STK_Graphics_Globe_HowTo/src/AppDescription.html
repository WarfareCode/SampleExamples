Name: CustomApp_AWT_STK_Graphics_Globe_HowTo
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the code needed to manipulate the graphics in the STKX Globe control (i.e. agi.stkx.awt.AgGlobeCntrlClass).
Directions:
  1. Locate the tree in the left pane of the application.
  2. Click on any node within the tree.
  3. Notice the graphics change in the Globe control.
  4. Notice the source code snippet provided in the botton pane of the application.
