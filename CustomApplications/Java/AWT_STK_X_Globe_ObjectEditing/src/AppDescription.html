Name: CustomApp_AWT_STK_X_Globe_ObjectEditing
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework.
Directions: 1. Press the Scenario -> New Scenario menu item.
2. Select the FacilityTo3DObjectEdit in the middle combo box.
3. Click the left most Object Editing Start button.
4. In the globe control left click with the mouse on one of the facility's (called FacilityTo3DObjectEdit) position vectors which will make it yellow and drag/move the facility in that direction to a new location.
5. Click the Object Editing Accept button to the right of the object path combo box above the globe control.
6. Press the Scenario -> Close Scenario menu item.
