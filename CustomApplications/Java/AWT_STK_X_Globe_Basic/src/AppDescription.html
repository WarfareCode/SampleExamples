Name: CustomApp_AWT_STK_X_Globe_Basic
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the basic code needed to embed the STKX Globe control (i.e. agi.stkx.awt.AgGlobeCntrlClass).
Directions:
  1. Press the Scenario -> New Scenario menu item.
  2. Using the mouse, left button click on the globe.
  3. Move the mouse left to right and top to bottom.
  4. Note the rotation movement of the globe.
  5. Release the mouse's left button.
  6. Using the mouse, right button click on the globe.
  7. Move the mouse top to bottom.
  8. Note the zoom in/out movement of the globe.
  9. Press the Scenario -> Close Scenario menu item.
