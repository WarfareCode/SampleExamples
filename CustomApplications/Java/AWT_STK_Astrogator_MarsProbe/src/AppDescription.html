Name: CustomApp_AWT_STK_Astrogator_MarsProbe
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the use of Astrogator's in modeling a Mars Probe Mission.
Directions:
  1. Click on the buttons in the right hand pane as they become enabled from top to bottom.
  2. Only click on the Close Scenario button if you wish to start over at any point.
  3. When you reach the Stop Near Mars notice the results of the Astrogator propagator are displayed in the message viewer panel in the lower left corner of the application.
  4. You can save or clear this message viewer's text if you wish by clicking on the message viewer toolbar at the left of the message viewer.
  5. Go to the source code of the application and view the SampleCode.java file if you are interested in how to code each button feature.
  6. Continue until you finally click the Close Scenario button.
