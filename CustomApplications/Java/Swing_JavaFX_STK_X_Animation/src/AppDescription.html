Name: CustomApp_Swing_JavaFX_STK_X_Animation
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Oracle AWT/Swing/JavaFX. It demonstrates the basic code needed to implement animation related activities.
Directions: 1. Press the Scenario -> New Scenario menu item.
2. Notice the JavaFX Toolbars and data displays that fade and translate in from the top and bottom of the main form.
3. Press toolbar buttons and notice the JavaFX button's custom animation.
4. Press the Scenario -> Close Scenario menu item.
5. Notice the JavaFX Toolbars and data displays that fade and translate out to the top and bottom of the main form.
