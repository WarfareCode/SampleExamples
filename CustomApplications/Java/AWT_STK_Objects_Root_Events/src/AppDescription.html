Name: CustomApp_AWT_STK_Objects_Root_Events
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the basic code needed to receive STK Object Root events.
Directions:
  1. Press the Scenario -> New Scenario menu item.
  2. Note the event messages in the event viewers at the bottom of the sample.
  3. Press the Tools -> Connect menu item.
  4. In the Connect dialog's text field, type New / Facility Fac1
  5. In the Connect dialog, click the Execute button.
  6. Note the event messages in the event viewers at the bottom of the sample.
  7. In the Connect dialog's text field, type Animate * Start
  8. In the Connect dialog, click the Execute button.
  9. Note the event messages in the message viewers at the bottom of the sample.
  10. In the Connect dialog's text field, type Animate * Pause
  11. In the Connect dialog, click the Execute button.
  12. Note the event messages in the message viewers at the bottom of the sample.
  13. In the Connect dialog's text field, type Unload / */Facility/Fac1
  14. In the Connect dialog, click the Execute button.
  15. Note the event messages in the message viewers at the bottom of the sample.
  16. Press the Scenario -> Close Scenario menu item.
