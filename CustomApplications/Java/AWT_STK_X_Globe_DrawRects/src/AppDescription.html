Name: CustomApp_AWT_STK_X_Globe_DrawRects
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the basic code needed to draw rectangles in the STKX Globe control (i.e. agi.stkx.awt.AgGlobeCntrlClass).
Directions:
  1. Press the Scenario -> New Scenario menu item.
  2. In the bottom panel, choose a Line Style in the first combo box.
  3. Then choose a Line Width in the second combo box.
  4. Then choose a Line Color in the button color chooser.
  5. Click the Add Rectangle button.
  6. Left click with the mouse on the globe control and drag down and to the right.
  7. Click the List All button.
  8. Click the Clear All button.
  9. Press the Scenario -> Close Scenario menu item.
