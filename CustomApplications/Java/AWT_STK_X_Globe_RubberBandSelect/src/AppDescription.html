Name: CustomApp_AWT_STK_X_Globe_RubberBandSelect
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the code needed to implement a "rubber band" to select objects and retrieve meta data on those objects in the STKX Globe control.
Directions: 1. Press the Scenario -> Load Scenario menu item.
2. Navigate to the ISR Scenario. It is located at {STK or STK Engine Install Dir}\CodeSamples\SharedResources\Scenarios\ISR\ISR.sc
3. Click the Select Objects button.
4. Left click and hold the left mouse button.
5. Drag the mouse to the desired opposite box corner point.
6. Release the left mouse button.
7. Press the Scenario -> Close Scenario menu item.
