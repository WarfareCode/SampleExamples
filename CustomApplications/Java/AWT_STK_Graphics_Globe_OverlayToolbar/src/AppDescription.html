Name: CustomApp_AWT_STK_Graphics_Globe_OverlayToolbar
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates how to add a Overlay Toolbar to the STKX Globe control (i.e. agi.stkx.awt.AgGlobeCntrlClass) using the CustomApp_AWT_STK_Graphics_Globe_Common project's OverlayToolBar.
Directions:
  1. Press the Scenario -> New Scenario menu item.
  2. Using the mouse, exercise the buttons on the Overlay Toolbar at the bottom of the Globe control.
  3. Press the Scenario -> Close Scenario menu item.
