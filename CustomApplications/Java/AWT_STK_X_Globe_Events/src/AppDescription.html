Name: CustomApp_AWT_STK_X_Globe_Events
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the basic code needed to receive STKX Globe control (i.e. agi.stkx.awt.AgGlobeCntrlClass) events.
Directions: 1. Press the Scenario -> New Scenario menu item.
2. Using the mouse, left button click on the globe.
3. Move the mouse left to right and top to bottom and note the rotation movement of the globe.
4. Release the mouse's left button.
5. Using the mouse, right button click on the globe.
6. Move the mouse top to bottom and note the zoom in/out movement of the globe.
7. Note the events in the message viewers.
8. Press the Scenario -> Close Scenario menu item.
