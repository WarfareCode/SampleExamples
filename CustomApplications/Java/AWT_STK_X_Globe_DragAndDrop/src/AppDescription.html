Name: CustomApp_AWT_STK_X_Globe_DragAndDrop
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the basic code needed to use the Microsoft OLE DragAndDrop feature in the STKX Globe control (i.e. agi.stkx.awt.AgGlobeCntrlClass).
Directions:
  1. Navigate to the Sample -> Drop Mode menu item and make sure Manual is checked/enabled.
  2. Locate the sample's project directory and expand the connect folder.
  3. Using the mouse, left click on the SetUp.connect file and drag it to the running application's Globe control.
  4. Using the mouse, left click on the AnimateStart.connect file and drag it to the running application's Globe control.
  5. Using the mouse, left click on the AnimatePause.connect file and drag it to the running application's Globe control.
  6. Using the mouse, left click on the AnimateReset.connect file and drag it to the running application's Globe control.
  7. Using the mouse, left click on the CloseScenario.connect file and drag it to the running application's Globe control.
