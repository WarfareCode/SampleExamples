Name: CustomApp_AWT_STK_X_TabPanes
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework.
Directions: 1. Create a new scenario by selecting the menu item of File -> New Scenario.
2. In the Tree Pane selected New Globe/VO/3D several times.
3. Notice that Globe tabs are created, note the window ids.
4. In the Tree Pane selected New Map/2D several times.
5. Notice that Map tabs are created, note the window ids.
6. Interact with the Globe and Map controls.
7. Issue connect commands by selecting the menu item of Tools -> Connect.
8. Close any combination of tabs of Globe and Map controls.
9. Close the scenario by selecting the menu item of File -> Close Scenario.
