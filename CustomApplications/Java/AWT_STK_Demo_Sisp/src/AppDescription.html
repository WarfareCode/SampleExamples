Name: CustomApp_AWT_STK_Demo_Sisp
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates a possible implementation of a Single Integrated Space Picture application.
Directions:
  1. In the bottom pane's Vehicle and Satellite tabs, multi-select a few objects in the "Available" table and click the "To The Right" button, moving the objects to the "Shown" table.
  2. In the Information pane on the left hand side of the application, click the Surveillance tab, in the Access group box click the Find Accesses check box.
  3. Notice a facility is displayed and accesses are computed as well as the access time range visually shown for each object's 3D/2D paths.
  4. Double click anywhere on the Globe control to change the location of the facility and recompute/revisualize accesses from that new globe location (lat, lon).
  5. Animate the scenario using the toolbar above the Globe/Map control tab pane.
  6. Notice when access is available from the facility to asset, a line is drawn connecting the two visually.
