Name: CustomApp_AWT_STK_X_Map_Projections
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the code needed to change the Map projection to EquiCylindrical, HaimerAitoff, etc.
Directions: 1. Press the Scenario -> New Scenario menu item.
2. Set the different options as is appropriate.
3. Click change projection button.
4. Press the Scenario -> Close Scenario menu item.
