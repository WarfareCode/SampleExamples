Name: CustomApp_AWT_STK_Objects_Tutorial_Begin
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing framework. It serves as beginning tutorial code with directions contained as TODO: comments in the samples java source files. Compare to CustomApp_AWT_STK_Objects_Tutorial_End when finished or if you need help.
Directions: 1. Complete the code sections that are marketed TODO: and get the application to compile.
2. Compare to CustomApp_AWT_STK_Objects_Tutorial_End if you need help or after completing it.
3. Click on the buttons in the right hand pane as they become enabled from top to bottom.
4. Only click on the Reset button if you wish to start over at any point.
5. Go to the source code of the application and view the SampleCode.java file if you are interested in how to code each button feature.
6. Continue until you finally click the Save/Unload Scenario button.
