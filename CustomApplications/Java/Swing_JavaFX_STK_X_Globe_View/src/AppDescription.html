Name: CustomApp_Swing_JavaFX_STK_X_Globe_View
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Engine Custom Application using Sun Microsystems's Java AWT/Swing/JavaFX framework. It demonstrates the code needed to change the view of the STKX Globe control.
Directions: 1. Press the Scenario -> New Scenario menu item.
2. Notice the JavaFX View Toolbar fades in and translates from the top of the UI.
3. Click the Zoom In button with image of a magnifying glass and plus sign.
4. Did you notice the custom animation of the Zoom In button?
5. Click the Home button with image of a house.
6. Did you notice the custom animation of the Home button?
7. Press the Scenario -> Close Scenario menu item.
