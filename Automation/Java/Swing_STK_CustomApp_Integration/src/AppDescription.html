Name: Automation_Swing_STK_CustomApp_Integration
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Automation API integrated with STK Engine API in one Application/process using Sun Microsystems's Java AWT/Swing framework. It demonstrates the basics of exporting a facility from STK Desktop and importing it into an STK Engine Application in the same process/application.
