Name: Automation_Swing_STK_Basic
Vendor: Analytical Graphics, Inc.
Description: This is an example of a STK Automation Application using Sun Microsystems's Java AWT/Swing framework. It demonstrates the basics of creating or attaching to an STK GUI application process and control it from a Java application process. In this sample only UI and a simple create/close scenarios and vdf's are demonstrated. The entire Object Model is accessible to the Java process.
Directions:
  1. Notice the status indicator label states DISCONNECTED and is red.
  2. Attach to an instance of STK...
    • If a current STK UI instance is not running, click Create New...
    • If a current STK UI instance is running, click Get Running...
  3. Notice the status indicator label states CONNECTED and is green.
  4. Choose the Scenario -> New Scenario menu item in the menu bar.
  5. Slightly change the Top, Left, Height, Width textfield values and click the Update button.
  6. Notice the size and position of the STK UI has changed.
  7. Click the Invisible radio button and click the Update button.
  8. Notice the STK UI application is now invisible, however, it still remains as an active process identified as AgUiApplication.exe in the Window's Task Manager.
  9. Click the Release button.
  10. Notice the STK UI process (AgUiApplication.exe) no longer exists in the Windows's Task Manager.
